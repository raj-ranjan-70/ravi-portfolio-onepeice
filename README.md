\## ⚙️ Technologies \& Tools



This project is forged with modern web technologies to ensure a seamless, high-performance experience:



\* \*\*React:\*\* Functional components and hooks for modular architecture.

\* \*\*Tailwind CSS:\*\* Utility-first styling for rapid, responsive design and custom theming.

\* \*\*Framer Motion:\*\* Powering complex, smooth-scrolling animations, glowing hover states, and staggered list reveals.

\* \*\*Modern UI:\*\* Fully responsive with dark theme support out-of-the-box.

\## 📂 Project Structure



A clean, component-based folder architecture:



```text

src/

├── assets/         # Images, icons, paper textures, fonts

├── components/     # Reusable UI (Navbar, BountyCard, TimelineItem, Footer)

├── pages/          # Main sections (Hero, Arsenal, Journey, Projects, Contact)

├── styles/         # Global styles and Tailwind configuration

├── App.jsx         # Root layout and scroll container

└── main.jsx        # React entry point

```

\## 🚀 Hoist the Colors (Getting Started)



To run this project locally and begin your adventure:



1\. \*\*Clone the repository:\*\*

```bash

git clone \[https://github.com/ravikantMishra10/portfolio.git](https://github.com/ravikantMishra10/portfolio.git)

```



2\. \*\*Navigate to the directory:\*\*

```bash

cd portfolio

```



3\. \*\*Install the dependencies:\*\*

```bash

npm install

```



4\. \*\*Set sail (Start the development server):\*\*

```bash

npm run dev

```
